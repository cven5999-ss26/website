# CVEN 5999: Special Topics - Data Analytics for Development

This repository contains the Quarto source for the Summer 2026 course website: <https://cven5999-ss26.github.io/website/>

The course is offered at the University of Colorado Boulder and taught by [Lars Schöbitz](https://www.linkedin.com/in/larsschoebitz/).

## Build

The site is built with [Quarto](https://quarto.org/). It renders to the `docs/` directory, which is served via GitHub Pages.

```bash
quarto render
```

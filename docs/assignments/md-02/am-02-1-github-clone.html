Clone a GitHub repository

Clone a repository

In this assignment, you will clone your personal Module 2 repository from GitHub to Posit Cloud.

  1. Open https://github.com/ in your browser. Login with your username if you are not logged in yet.

  1. Navigate to the GitHub organization for the course: https://github.com/cven5999-ss26

  1. Find the repository md-02-USERNAME that ends with your GitHub username, and open it by clicking on the repository name.
  • Replace USERNAME with your actual GitHub username.
  • For example, if your username is johnsmith, the repository will be md-02-johnsmith.
Tip

You can search for your repository by typing your username in the search bar just below the Repository heading.

  1. Click on the green Code button.

  1. Copy the HTTPS URL to your clipboard by clicking on the clipboard icon next to the URL.

  1. Open the Content tab of the cven5999-ss26 workspace on Posit Cloud: https://posit.cloud/spaces/784454/content/all?sort=name_asc

  1. Click the blue button New Project > New Project from Git Repository

  1. Paste the HTTPS URL from GitHub into the URL of your Git Repository field and click OK. Leave the checkbox under Packages checked.

  1. Wait until the project is deployed. This may take a few minutes, depending on your internet connection.

  1. Continue with the next assignment: GitHub workflow.
